# 📱 Smartphone Usage and Addiction Level Analysis  

## 📖 Overview  
This project analyzes smartphone usage patterns and identifies addiction levels among users. With the rapid increase in smartphone usage, it is important to understand its impact on **sleep, stress, productivity, and lifestyle**.

The project uses a dataset of **7500 records** and presents insights through an **interactive Excel dashboard**.

---

## 🎯 Objectives  
- Analyze smartphone usage behavior  
- Identify addiction levels (None, Mild, Moderate, Severe)  
- Study the relationship between screen time and sleep  
- Examine the impact on stress levels  
- Compare usage across age and gender  

---

## 📊 Dataset  
The dataset includes the following features:

- Age  
- Gender  
- Daily Screen Time  
- Weekend Screen Time  
- Social Media Hours  
- Gaming Hours  
- Work/Study Hours  
- Sleep Hours  
- Stress Level  
- Addiction Level

## 🔍 Key Insights  
- 📉 Increased screen time reduces sleep quality  
- 📱 Social media is a major cause of addiction  
- 🎮 Gaming contributes to severe addiction  
- 😟 Higher usage leads to increased stress  
- 👩‍🎓 Younger users show higher addiction levels  

---

## 📌 Conclusion  
The analysis shows that excessive smartphone usage negatively affects **mental health, sleep, and productivity**. Managing screen time is important for maintaining a balanced lifestyle.
